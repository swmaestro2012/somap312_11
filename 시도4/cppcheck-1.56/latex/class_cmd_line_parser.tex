\subsection{Detailed Description}
The command line parser. The command line parser parses options and parameters user gives to cppcheck command line. 

The parser takes a pointer to \hyperlink{class_settings}{Settings} instance which it will update based on options user has given. Couple of options are handled as class internal options.
