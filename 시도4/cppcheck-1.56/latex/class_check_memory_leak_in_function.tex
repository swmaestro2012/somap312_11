\subsection{Detailed Description}
Check\-Memory\-Leak\-In\-Function detects when a function variable is allocated but not deallocated properly. 

The checking is done by looking at each function variable separately. By repeating these 4 steps over and over\-:
\begin{DoxyEnumerate}
\item locate a function variable
\item create a simple token list that describes the usage of the function variable.
\item simplify the token list.
\item finally, check if the simplified token list contain any leaks. 
\end{DoxyEnumerate}
