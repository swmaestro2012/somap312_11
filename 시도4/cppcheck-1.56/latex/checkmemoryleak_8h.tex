\subsection{Detailed Description}
Check for memory leaks

The checking is split up into three specialized classes.
\begin{DoxyItemize}
\item \hyperlink{class_check_memory_leak_in_function}{Check\-Memory\-Leak\-In\-Function} can detect when a function variable is allocated but not deallocated properly.
\item \hyperlink{class_check_memory_leak_in_class}{Check\-Memory\-Leak\-In\-Class} can detect when a class variable is allocated but not deallocated properly.
\item \hyperlink{class_check_memory_leak_struct_member}{Check\-Memory\-Leak\-Struct\-Member} checks allocation/deallocation of structs and struct members 
\end{DoxyItemize}
