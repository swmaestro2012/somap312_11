Extract the code for each configuration. Use this with getcode() to get the file data for each individual configuration.

Parameters
srcCodeStreamThe (file/string) stream to read from.
processedFileGive reference to empty string as a parameter, function will fill processed file here. Use this also as a filedata parameter to getcode() if you received more than once configurations.
resultConfigurationsList of configurations. Pass these one by one to getcode() with processedFile.
filenameThe name of the file to check e.g. "src/main.cpp"
includePathsList of paths where incude files should be searched from, single path can be e.g. in format "include/". There must be a path separator at the end. Default parameter is empty list. Note that if path from given filename is also extracted and that is used as a last include path if include file was not found from earlier paths.
