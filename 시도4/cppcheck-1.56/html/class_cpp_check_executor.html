Detailed Description

This class works as an example of how CppCheck can be used in external programs without very little knowledge of the internal parts of the program itself. If you wish to use cppcheck e.g. as a part of IDE, just rewrite this class for your needs and possibly use other methods from CppCheck class instead the ones used here.
